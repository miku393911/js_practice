# JavaScript 基礎問題集 vol.4

JavaScriptの基本文法（if, else, for, while, 配列, 関数）に関する練習問題です。
**制約事項**: `sort`, `Math.max`, `Math.min`, `join`, `reverse`, `map`, `filter` などの便利なメソッドや、`Set` などのオブジェクトは**使用せず**に、基本的な `for` ループや `if` 文などの構文のみを使って解いてください。

---

### 第1章：条件分岐 (if, else)

1. 変数 `num` が正の数なら「正」、負の数なら「負」、0なら「ゼロ」と出力してください。
2. 変数 `num` の絶対値を計算して出力してください。（`Math.abs` は使用禁止。if文で実現すること）
3. 3つの変数 `a`, `b`, `c` の中から最も大きい数値を特定して出力してください。（`Math.max` は使用禁止）
4. 変数 `year` がうるう年かどうかを判定して出力してください。うるう年は「4で割り切れる年はうるう年。ただし100で割り切れる年は平年。ただし400で割り切れる年はうるう年」とします。
5. 変数 `score` (0-100) に応じて成績を出力してください。90以上: "S", 80以上: "A", 60以上: "B", 60未満: "C"。
6. 変数 `n` が 3の倍数なら "Fizz"、5の倍数なら "Buzz"、両方の倍数なら "FizzBuzz"、それ以外ならその数値を出力してください。

### 第2章：繰り返し (for)

7. for文を使って、1 から 50 までの整数の総和を計算して出力してください。
8. for文を使って、1 から 30 までの整数のうち、奇数だけを順番に出力してください。
9. 変数 `n` (例: 5) の回数分、文字列 "*" を連結して "*****" のような文字列を作成して出力してください。（`repeat` メソッド禁止）
10. for文を使って、7 の段の九九（7*1=7 から 7*9=63 まで）を計算式とともに出力してください。
11. 変数 `n` (例: 5) の階乗（5! = 5*4*3*2*1）を計算して出力してください。
12. 初項 3、公差 4 の等差数列を、値が 100 を超えない範囲で順に出力してください。

### 第3章：繰り返し (while)

13. 変数 `money` (例: 1000) が 0 になるまで、毎回 100 ずつ減らしながら現在の残高を出力してください。
14. 変数 `n` (正の整数) に対して、偶数なら 2 で割り、奇数なら 3 倍して 1 を足す操作を繰り返し、最終的に 1 になるまでの過程（数値）を出力してください（コラッツの予想）。
15. 変数 `n` (例: 12345) の「桁数」を、文字列変換せずに while 文と割り算を使って求めて出力してください。
16. 2 を繰り返し掛けていき（2, 4, 8...）、値が 1000 を超えた時点の数値を出力してください。
17. 変数 `n` が素数かどうかを判定するプログラムを書いてください。2から `n-1` までの数で割り切れるかを確認し、「素数です」または「素数ではありません」と出力してください。
18. 変数 `a` と `b` の最大公約数を、ユークリッドの互除法（while文）を使って求めて出力してください。

### 第4章：配列 (Array)

※ この章では、`forEach`, `map`, `filter`, `reduce`, `find`, `indexOf` などの高階関数やメソッドは**禁止**です。すべて `for` 文とインデックス (`i`) で操作してください。

19. 配列 `numbers` (例: `[10, 25, 4, 30]`) の要素をすべて足し合わせた合計を出力してください。
20. 配列 `numbers` の中から最大値を見つけて出力してください。
21. 配列 `numbers` の中に、特定の数値 `target` (例: 5) が何個含まれているかを数えて出力してください。
22. 配列 `numbers` の要素を逆順にした新しい配列 `reversed` を作成して出力してください。（`reverse`禁止）
23. 2つの配列 `arr1` と `arr2` を結合した新しい配列 `combined` を作成してください。（`concat`禁止）
24. 配列 `numbers` に含まれる重複した要素を取り除き、ユニークな値だけの新しい配列を作成してください。（`Set`禁止、ループで確認すること）

### 第5章：関数 (Function)

25. 底辺 `base` と高さ `height` を受け取り、三角形の面積を返す関数 `getTriangleArea` を作成してください。
26. 正の整数 `n` を受け取り、それが素数であれば `true`、そうでなければ `false` を返す関数 `isPrime` を作成してください。
27. 配列を受け取り、その中の偶数の数（個数）を返す関数 `countEvens` を作成してください。
28. 2つの整数 `start` と `end` を受け取り、`start` から `end` までの整数の総和を返す関数 `sumRange` を作成してください。
29. 配列を受け取り、その平均値を計算して返す関数 `getAverage` を作成してください。
30. 数値 `base` と 指数 `exponent` を受け取り、`base` の `exponent` 乗を計算して返す関数 `power` を作成してください。（`Math.pow`, `**` 演算子禁止。ループで実装すること）
