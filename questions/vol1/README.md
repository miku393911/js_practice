# JavaScript 実践練習問題 Vol 1

このリポジトリの Volume 2 以前に作成された、JavaScript の基礎固めのための練習問題集です。

## コンテンツ一覧
Volume 1 は以下の 2 部構成になっています。

1. **[Part 1: 基礎・ロジック編](./JavaScript_practice01.md)**
    - FizzBuzz
    - 配列とループによる最大値検索
    - switch 文によるステータス判定
    - ショッピングカートの集計
    - クラスと継承の基礎
    - 配列メソッド (map, filter, find) の基礎
    - コールバック関数
    - ファイル分割 (import / export)

2. **[Part 2: 総合演習編](./JavaScript_practice02.md)**
    - 変数と計算、テンプレートリテラル
    - Truthy/Falsy の理解
    - スコープの理解
    - 配列・オブジェクトの詳細操作
    - メソッドチェーン
    - バリデーションロジック
    - クラスの応用
    - Todoリスト管理クラスの作成

それぞれのリンク先の `.md` ファイルにある問題を確認し、`questions/vol1/` 配下の `.js` ファイルで実装を行ってください。
