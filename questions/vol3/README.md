# JavaScript 基礎問題集 vol.3

JavaScriptの基本文法（if, else, for, while, 配列, 関数）に関する練習問題です。
メソッド（sort, join, Math.maxなど）を使わずに、基本的な構文のみを使って解いてください。

---

### 第1章：条件分岐 (if, else)

1. 変数 `num` が 10 より大きい場合に、「10より大きい」とコンソールに出力してください。
2. 変数 `num` が 0 以上なら「正またはゼロ」、そうでなければ「負の数」と出力してください。
3. 変数 `score` が 80 以上なら「A」、60 以上 80 未満なら「B」、それ以外なら「C」と出力してください。
4. 変数 `num` が偶数（2で割り切れる）の場合に、「偶数です」と出力してください。
5. 変数 `str` が空文字列 `""` である場合に、「空です」と出力してください。
6. 変数 `age` が 20 以上なら「成人」、そうでなければ「未成年」と出力してください。

### 第2章：繰り返し (for)

7. for文を使って、1 から 10 までの数値を順番に出力してください。
8. for文を使って、10 から 1 までの数値を逆順に出力してください。
9. for文を使って、1 から 20 までの数値のうち、偶数だけを出力してください。
10. for文を使って、1 から 5 までの数値の合計（1+2+3+4+5）を計算し、最後に出力してください。
11. for文を使って、文字列 「JavaScript」 を 5 回出力してください。
12. for文を使って、1 から 100 までの数値のうち、5 の倍数だけを出力してください。

### 第3章：繰り返し (while)

13. while文を使って、変数 `count` が 5 未満の間、`count` の値を出力し、`count` を 1 ずつ増やしてください。
14. while文を使って、変数 `n` が 1 から始まり、100 を超えるまで `n` を 2 倍にし続け、各ステップの `n` を出力してください。
15. while文を使って、変数 `i` が 10 から始まり、0 より大きい間、`i` を 2 ずつ減らして出力してください。
16. while文を使って、1 から順に数値を足していき、合計が 20 を超えた時点での「最後に足した数値」を出力してください。
17. 変数 `isCorrect` が `false` の間ループし続け、ループ内で特定の条件（例：iが3になったら）で `isCorrect` を `true` にしてループを抜ける処理を書いてください。
18. while文を使って、3 の倍数を 10 から逆順に探し、最初に見つかった 3 の倍数（9）を出力して終了する処理を書いてください。

### 第4章：配列 (Array)

19. 配列 `const arr = [10, 20, 30];` の 2 番目（インデックス 1）の要素をコンソールに出力してください。
20. 配列 `const items = ["apple", "orange"];` の最初の要素を "banana" に書き換えてください。
21. for文を使って、配列 `[1, 2, 3, 4, 5]` のすべての要素を順番に出力してください。
22. for文を使って、配列 `[10, 20, 30]` の要素の合計を計算し、出力してください。
23. 配列 `[1, 5, 8, 10, 15]` の中から、10 以上の数値だけをループで取り出して出力してください。
24. 配列 `const data = [1, 2, 3, 4];` の要素の数を、`length` プロパティを使って出力してください。

### 第5章：関数 (Function)

25. 2つの引数を受け取り、その和を返す関数 `add` を定義してください。
26. 引数 `name` を受け取り、「こんにちは、[name]さん」と出力する関数 `sayHello` を定義してください。
27. 1つの引数を受け取り、その数が偶数なら `true`、奇数なら `false` を返す関数 `isEven` を定義してください。
28. 配列を引数として受け取り、その配列の「最初の要素」を返す関数 `getFirst` を定義してください。
29. 引数 `n` を受け取り、1 から `n` までの合計を計算して返す関数 `totalSum` を定義してください。
30. 引数 `price` を受け取り、消費税 10% を加えた数値（price * 1.1）を返す関数 `getTaxIncluded` を定義してください。
