# JavaScript 基礎問題集 vol.5

JavaScriptの基本文法（ループ、条件分岐、配列操作）に関する練習問題です。
このボリュームでは、数学的な知識（素数、階乗など）を必要とせず、日常生活やアプリ開発でよくある具体的なロジックを扱います。

**制約事項**: 基本的な `for` ループ、`while` ループ、`if` 文などの構文を使って解いてください。

---

### 第1章：繰り返しと条件分岐

1. **カウントダウン**: 10から1まで、1ずつ減らしながら数値を出力するプログラムを書いてください。
2. **倍数の合計**: 1から20までの数のうち、「3の倍数」だけをすべて足して、その合計を出力するプログラムを書いてください。
3. **挨拶のループ**: 名前が入った配列 `["田中", "佐藤", "鈴木"]` を使い、全員分「こんにちは、〇〇さん」と出力するプログラムを書いてください。
4. **配列内検索と中断**: 果物が入った配列 `["みかん", "バナナ", "りんご", "ぶどう"]` の中に「りんご」があるか探し、見つけたら「見つけました！」と表示して、その瞬間にループを終了するプログラムを書いてください。
5. **合計値の制限**: 1, 2, 3...と順番に数値を足していき、合計が 50 を超えた瞬間の合計値を出力するプログラムを書いてください。

### 第2章：繰り返し (for)

6. **指定回数の繰り返し**: for文を使って、"Hello!" と 5回出力するプログラムを書いてください。
7. **総和の計算**: for文を使って、1 から 100 までの整数の合計を計算して出力してください。
8. **特定の範囲の出力**: for文を使って、10 から 20 までの整数を順番に出力してください。
9. **偶数のみ出力**: for文を使って、1 から 20 までの整数のうち、偶数（2で割り切れる数）だけを出力してください。
10. **配列の要素を出力**: 配列 `colors = ["Red", "Green", "Blue"]` のすべての要素を for文を使って順番に出力してください。
11. **配列の合計**: 数値が入った配列 `numbers = [10, 20, 30, 40, 50]` の合計を for文を使って計算してください。
12. **特定の文字を数える**: 文字列 `"programming"` の中に、文字 `"g"` がいくつ含まれているか数えるプログラムを書いてください。

### 第3章：繰り返し (while)

13. **条件によるループ**: 変数 `num` が 100 未満の間、`num` を 2倍にし続けるプログラムを書いてください。（初期値は `1` とします）
14. **カウントダウン**: 変数 `count` を 5 から 1 まで減らしながら出力し、最後に "Start!" と出力するプログラムを書いてください。
15. **残高シミュレーション**: 変数 `balance`（残高）を 1000 とし、ループするたびに 150 ずつ減らしていき、残高がマイナスになる手前までの各残高を出力してください。
16. **特定の値が出るまで**: 配列 `nums = [1, 3, 5, 7, 8, 9]` を先頭から順に見ていき、偶数（この場合は 8）が見つかったら「見つけました！」と出力してループを終了してください。
17. **入力チェック**: 配列 `inputs = ["no", "no", "yes", "no"]` を順にチェックし、"yes" が出たら「承認されました」と出力して処理を終了してください。
18. **文字数チェック**: 配列 `words = ["apple", "banana", "kiwi", "grape"]` から、文字数が 5文字以上の単語が見つかるまで順に出力し続けてください。

### 第4章：配列 (便利なメソッド)

※ この章に限り、`map`, `filter`, `find`, `includes` などの**配列メソッドを使用することを推奨**します！（for文を使わずにスマートに書いてみましょう）

19. **すべての要素を2倍にする**: 配列 `[1, 2, 3, 4, 5]` のすべての要素を 2倍にした新しい配列を `map` を使って作成し、出力してください。
20. **条件に合うものだけ残す**: 配列 `[10, -5, 20, -3, 30]` から、正の数（0より大きい数）だけを集めた新しい配列を `filter` を使って作成し、出力してください。
21. **最初のひとつを見つける**: 配列 `[10, 15, 20, 25]` の中から、最初に 20 以上になる数値を見つけて `find` を使って出力してください。
22. **含まれているかチェック**: 配列 `["cat", "dog", "bird"]` の中に "dog" が含まれているかを `includes` を使って確認し、`true` または `false` を出力してください。
23. **文字をつなげる**: 配列 `["Japan", "USA", "UK"]` の要素を `" / "` という区切り文字でつないで、1つの文字列として `join` を使って出力してください。
24. **合計を計算する（応用）**: 配列 `[10, 20, 30]` の合計を `reduce` を使って計算してみてください。（難しければ for文でもOKですが、ぜひ `reduce` に挑戦してみてください）

### 第5章：関数 (Function)

25. **挨拶関数**: 名前を引数 `name` として受け取り、"こんにちは、`name` さん！" という文字列を返す関数 `greet` を作成してください。
26. **計算関数**: 2つの数値 `a` と `b` を受け取り、その掛け算の結果を返す関数 `multiply` を作成してください。
27. **偶数判定**: 数値 `n` を受け取り、それが偶数なら `true`、奇数なら `false` を返す関数 `isEven` を作成してください。
28. **最大値判定**: 2つの数値を受け取り、大きい方の数値を返す関数 `getMax` を作成してください。（`Math.max` を使ってもOKです）
29. **配列の平均**: 数値の配列を受け取り、その平均値を計算して返す関数 `calculateAverage` を作成してください。
30. **パスワード文字数チェック**: 文字列 `password` を受け取り、長さが 8文字以上なら `true`、そうでなければ `false` を返す関数 `isValidPassword` を作成してください。
