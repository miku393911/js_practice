# JavaScript 基礎問題集 vol.5

JavaScriptの基本文法（ループ、条件分岐、配列操作）に関する練習問題です。
このボリュームでは、数学的な知識（素数、階乗など）を必要とせず、日常生活やアプリ開発でよくある具体的なロジックを扱います。

**制約事項**: 基本的な `for` ループ、`while` ループ、`if` 文などの構文のみを使って解いてください。高階関数（`forEach`, `map`, `filter` など）は**使用せず**に実装してみましょう。

---

### 第1章：繰り返しと条件分岐

1. **カウントダウン**: 10から1まで、1ずつ減らしながら数値を出力するプログラムを書いてください。
2. **倍数の合計**: 1から20までの数のうち、「3の倍数」だけをすべて足して、その合計を出力するプログラムを書いてください。
3. **挨拶のループ**: 名前が入った配列 `["田中", "佐藤", "鈴木"]` を使い、全員分「こんにちは、〇〇さん」と出力するプログラムを書いてください。
4. **配列内検索と中断**: 果物が入った配列 `["みかん", "バナナ", "りんご", "ぶどう"]` の中に「りんご」があるか探し、見つけたら「見つけました！」と表示して、その瞬間にループを終了するプログラムを書いてください。
5. **合計値の制限**: 1, 2, 3...と順番に数値を足していき、合計が 50 を超えた瞬間の合計値を出力するプログラムを書いてください。
