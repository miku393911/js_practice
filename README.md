# JavaScript Practice Repository

このリポジトリは、JavaScript の文法確認と実践的なコーディングスキルを磨くための練習問題を集めたものです。

## 構成

各練習問題は `questions` フォルダ配下にボリュームごとに整理されています。

### 📚 練習問題一覧

| ボリューム | 内容 | リンク |
| :--- | :--- | :--- |
| **Vol 1** | 基本文法、ループ、クラス、モジュール化 | [問題を見る](./questions/vol1/README.md) |
| **Vol 2** | ES6+ モダン文法、配列メソッド、モダン演算子 | [問題を見る](./questions/vol2/README.md) |
| **Vol 3** | 基本文法（if, for, while, 配列, 関数） | [問題を見る](./questions/vol3/README.md) |
| **Vol 4** | 基本文法復習（メソッド禁止縛り） | [問題を見る](./questions/vol4/README.md) |

---

## 🚀 使い方

### 1. 準備
リポジトリをクローンした後、依存関係をインストールしてください（必要な場合）。
```bash
npm install
```

### 2. 問題を確認する
上記のリンクから、各ボリュームの `README.md` を開き、解きたい問題を確認してください。

### 3. コードを書く・実行する
各ボリュームのフォルダ内に解答用の `.js` ファイルを用意しています。
エディタでコードを記述し、以下のコマンドで実行して結果を確認してください。

```bash
# 実行例 (Vol 2 の Q1)
node questions/vol2/q01_addArrow.js
```
